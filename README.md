# CLX Meta Dashboard

Sistema de Business Intelligence corporativo para la gestión de metas mensuales, facturación diaria y sincronización de tasas BCV.

## Características
- Gestión de Sucursales y Metas.
- Sincronización automática de tasas USD/EUR (BCV).
- Reportes profesionales en PDF y Excel.
- Integración nativa con Google Sheets.

Desarrollado con Streamlit.
